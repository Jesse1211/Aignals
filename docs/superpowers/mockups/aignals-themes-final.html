Theme persists to AignalsConfig.theme (new field, defaults to glassDark). Switching is instant — the whole panel re-styles via a single Theme enum threaded through the views. Glass Light/Dark use NSVisualEffectView; Terminal/Vibrant are fixed palettes. The menu-bar icon is unaffected (macOS template constraints). Animations are a later pass.
