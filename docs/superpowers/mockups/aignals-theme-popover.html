Aignals — Theme as a pop-out card

Click "Theme ▸" in Settings → a card slides out to the side, one theme per row (with a live swatch). Two side options below.

A · slides out RIGHT
Click "Theme" and the card extends to the right of the panel, like a native submenu. Each row: color swatch + name + ✓ on the active one.

SwiftUI: the "Theme" row is a button that toggles a side .popover(arrowEdge:) (or a custom overlay card). Each row carries a live mini-swatch so you preview the theme before picking; the active theme shows a ✓. Selecting re-styles the whole panel instantly and saves to AignalsConfig.theme. Slide-in animation can come in the animation pass.
