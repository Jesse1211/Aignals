Glass = NSVisualEffectView (.hudWindow / .popover material) behind a SwiftUI overlay — it follows the system appearance automatically. Signal dots get a colored halo via .shadow. Rows are cards with hover highlight, rounded corners, and trailing controls that fade in on hover. Animations are a separate pass.
